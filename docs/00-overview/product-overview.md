# Product Overview

## What Rydnex Is

Rydnex is a luxury-themed React Native vehicle management app built with Expo and Expo Router.
It helps individual vehicle owners track:

- Vehicle profiles
- Odometer history
- Fuel logs and basic efficiency metrics
- Service and repair history
- Recurring vehicle checks
- Parts-life replacement reminders

## Target Experience

The current app emphasizes:

- Fast daily logging actions
- A premium dark visual style
- Low-friction local usage
- Basic free-tier limits for growth gating

## Platforms

- iOS
- Android
- Web (with platform-specific layout adjustments in some screens)

## Current Feature Set (Implemented)

- Auth-like local sign-in flow (email validation + local session persistence)
- Garage management with free-tier vehicle cap (2 vehicles)
- Dashboard with vehicle-specific metrics and quick actions
- Logbook timeline (service, fuel, odometer)
- Last checks tracking (oil, tyre pressure, coolant, spare wheel, lights)
- Parts reminders with warning/due status
- Profile and account edit flow

## Planned/Placeholder Areas

The Profile experience includes explicit placeholders for:

- Insurance partner integrations
- Mechanic partner integrations
- Vehicle valuation
- Cloud backup
- Fleet management
